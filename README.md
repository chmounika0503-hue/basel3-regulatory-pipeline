
# Basel III Regulatory Compliance Pipeline

Tech Stack:
Python, PySpark, PostgreSQL, Airflow, Plotly Dash, Docker

Pipeline:
CSV Balance Sheets -> PySpark Processing -> Basel III Metrics -> PostgreSQL -> Airflow -> Dashboard

Metrics:
- CAR (Capital Adequacy Ratio)
- LCR (Liquidity Coverage Ratio)
- NPL Ratio (Non Performing Loan Ratio)
